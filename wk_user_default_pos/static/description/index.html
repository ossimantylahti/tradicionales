Restrict User To Default POS

This module restricts the user to his default POS while the other POS(s) are not visible to him.

Features:

  • POS Manager can assign default Point Of Sale to the other POS users.
  • POS Users would have the access to only their default Point Of Sale.
  • POS Manager has access to all the Point Of Sales.
