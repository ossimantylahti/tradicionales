Point of Sales with Customer Wallet Payment Management.

If you are using a POS and want to add balance to your customer wallet then this will be the good module for use. By using this module you can easily add/update wallet balance of your customer . This wallet can be used as Payment on time of POS payment which special POS Wallet journal. Customer wallet balance can easily shown on each customer on POS screen , So don't need to go back and check each customer's wallet balance, When any order made that customer's wallet balance will shown in front of cashier. So easily choose whether wants to pay from Wallet Balance (partially or fully) or pay from card/cash (normal payment method of odoo).

Click here for the video tutorial

Key features:

  • Easy to Add/Update wallet balance amount of each customer from both backend & POS.
  • Customer can Pay fully or partially Generated POS order by using his/her wallet balance easily.
  • Wallet Transaction history will be Generated for both credit and debit transaction of both backend & POS.
  • Customer can not pay more then his/her available credit wallet balance from wallet payment journal.
  • Customer can pay order from multiple payment method together when using Wallet payment option.
  • When Customer used wallet balance . That wallet balance of particular customer will be automatically updated.
  • Correct Journal entry has been generated with wallet balance option as its generated with default Odoo POS. So no need to worry about accounting transaction effected by this module.
  • No Complex Configuration.
